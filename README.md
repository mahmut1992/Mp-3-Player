# 🏠Mp3-Player

In this repository, a mini MP3 player has been designed using HTML, CSS, and JavaScript, allowing users to easily listen to their favorite songs, skip tracks, shuffle, and rewind.

# 🚀Features

-Responsive Design

-User-Friendly Interface

-Modular Coding

# 🛠️Technologies Used

-Html

-Css

-JavaScript
